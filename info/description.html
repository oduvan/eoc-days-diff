How old are you in number of days? It's easy to calculate - just subtract your birthday from today. We could make this a real challenge though and count the difference between any dates.

You are given two dates as array with three numbers - year, month and day. For example: 19 April 1982 will be (1982, 4, 19). You should find the difference in days between the given dates. For example between today and tomorrow = 1 day. The difference will always be either a positive number or zero, so don't forget about absolute value.

Input: Two dates as {% if interpreter.slug == "js-node" %}arrays of integers{% else %}tuples of integers{% endif %}.

Output: The difference between the dates in days as an integer.

Example:

{% if interpreter.slug == "js-node" %} 
daysDiff((1982, 4, 19), (1982, 4, 22)) == 3
daysDiff((2014, 1, 1), (2014, 8, 27)) == 238
{% else %} 
days_diff((1982, 4, 19), (1982, 4, 22)) == 3
days_diff((2014, 1, 1), (2014, 8, 27)) == 238
{% endif %}

Precondition: Dates between 1 january 1 and 31 december 9999. Dates are correct.
